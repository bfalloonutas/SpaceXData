For Space Enthusiasts

Purpose

The purpose of this project is to give easy access to data from the public spaceX api for space enthusiasts to enjoy.

Features

the pages on this is site display the following information from the SpaceX API

Launches are ordered by launch date:

  • Flight number
  • Flight name
  • Launch date
  • Any payloads the flight carried
  • Details of the flight if provided

Crew are shown in alphabetical order:

  • Name
  • Flight on which they were a crew member
  • A link to their wikipedia page

Payloads are shown in alphabetical order:

  • Name
  • Type
  • details of the flight it launched on
  • manufacturers
